# URL Shortener with MongoDB

This is a simple URL shortener built with Node.js, Express, MongoDB, and EJS.

## 🚀 Features

- Shorten long URLs to short codes
- Redirect using generated short codes
- Stores data in MongoDB

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
